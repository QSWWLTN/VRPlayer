请从 gde_gozen releases 下载对应平台的二进制文件:
https://github.com/VoylinsGamedevJourney/gde_gozen/releases

或从 Ko-fi / itch.io 获取编译好的版本:
https://ko-fi.com/s/c6ec85052b
https://voylin.itch.io/gde-gozen-video-playback-addon-for-godot

需要放入此目录的二进制文件:
- Windows: libgozen.windows.template_*.dll + FFmpeg DLLs
- Linux:   libgozen.linux.template_*.so + FFmpeg .so
- Android: libgozen.android.template_*.so + FFmpeg .so
- macOS:   libgozen.macos.template_*.dylib + FFmpeg .dylib
